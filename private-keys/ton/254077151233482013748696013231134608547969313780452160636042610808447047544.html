TON Private Keys

TON (The Open Network) uses Ed25519 for cryptography. Addresses are derived from wallet smart contracts, with different versions (V3R2, V4R2, W5) producing different addresses from the same key.

Unlike UTXO chains, TON wallets are smart contracts - the address depends on both the key and wallet version.
