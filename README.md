# PKF - Private Key Finder Website

Static website files for privatekeyfinder.io

## Deployment

### cPanel Hosting
1. Login to cPanel → File Manager
2. Go to `public_html` folder
3. Upload all files from this folder
4. Done! Site is live.

### GitHub Pages
1. Push this folder to GitHub repository
2. Go to repo Settings → Pages
3. Select "Deploy from branch" → main → / (root)
4. Done! Site is live at `https://username.github.io/repo-name`

### Netlify
1. Go to https://app.netlify.com/drop
2. Drag this folder to the drop zone
3. Done! Site is live.
