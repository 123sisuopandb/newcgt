Paper Wallet Generator

Generate secure, printable cryptocurrency wallets. 100% client-side generation.

Security: Print offline if possible. Store securely. Never photograph or scan with internet-connected devices. The private key controls all funds.

Click "Generate" to create your paper wallet

How to Use
Receive

Share the PUBLIC ADDRESS or its QR code. Anyone can send crypto to this address.

Spend

Import the PRIVATE KEY into a wallet app to access funds. Move all funds after importing.

Protect

Store in a safe, dry place. Consider laminating. Make backup copies stored separately.

FAQ

All keys are generated locally in your browser using crypto.getRandomValues(). No data is sent to any server. For maximum security: disconnect from the internet, use incognito mode, clear browser data after printing.

SegWit (bc1q): Recommended for Bitcoin - lower fees, wide support.
Legacy (1...): Maximum compatibility with all wallets.
Taproot (bc1p): Latest Bitcoin format with privacy benefits.
EVM (0x): Ethereum, BNB Chain, Polygon, Arbitrum, etc.
