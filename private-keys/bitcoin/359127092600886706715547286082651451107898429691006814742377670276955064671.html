Bitcoin Private Keys

Bitcoin uses ECDSA with the secp256k1 curve. Each private key generates multiple address formats: Legacy (P2PKH starting with 1), SegWit (P2WPKH starting with bc1q), Nested SegWit (P2SH starting with 3), and Taproot (P2TR starting with bc1p).

The private key can be expressed in WIF (Wallet Import Format) for easier backup - compressed keys start with K or L, uncompressed with 5.
