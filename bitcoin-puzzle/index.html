What is Bitcoin Puzzle?

The Bitcoin Puzzle (also known as the "1000 BTC Challenge") was created in 2015 by an anonymous user who distributed approximately ~1000 BTC across 160 addresses. Each address contains a private key with incrementally increasing bit lengths, making them progressively harder to crack.

To solve a puzzle, you need to find the private key within a specific range using brute-force or advanced cryptographic methods. Currently 82 puzzles have been solved, with the largest unsolved puzzles containing significant BTC rewards.
